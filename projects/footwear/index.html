Footwear e-commerce platform built with Next.js, Contentful, and Supabase.

Vercel Deployment GitHub Repository

Key Features:

  • Responsive design.
  • Options for both user and guest checkout.
  • Filtering system to refine search by brand and size, and sort by price or new arrivals.
  • Product search using keywords.
  • Identification of low-stock and out-of-stock items.
  • User account section for profile management, order tracking, and saving favorite items to a personalized wishlist.
