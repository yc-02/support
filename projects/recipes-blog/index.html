Recipes Blog built with Next.js and Supabase.

Vercel Deployment GitHub Repository

Key Features:

  • Responsive design.
  • Email and username validation in the sign-up form to ensure uniqueness.
  • Sign-up options include email and Google account.
  • Full control over recipes, allowing users to add, edit, or delete them.
  • Personalization of profiles by updating name, username, and profile pictures.
  • Recipe search using keywords and sorting by popularity, recency, and cooking time.
  • Ability to save recipes as favorites and explore other users' blogs.
