Travel Blog built with Node.js, Express.js, and MongoDB.

Vercel Deployment GitHub Repository

Key Features:

  • Responsive design.
  • Secure sign-up and sign-in using JSON Web Tokens (JWT).
  • Email and username validation in the sign-up form to ensure uniqueness.
  • Secure storage of user information and blog posts in MongoDB.
  • Full control over posts, allowing users to add, edit, or delete them.
  • The add and edit forms dynamically adjust textarea fields based on the number of days entered, streamlining the process of entering travel details for users on the travel blog.
