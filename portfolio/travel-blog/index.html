Travel Blog built with Node.js, Express.js, and MongoDB.

Vercel Deployment GitHub Repository

Key Features:

  • Developed backend logic with Node.js and Express.js, ensuring robust and scalable API endpoints.
  • Utilized MongoDB for secure data storage and efficient data retrieval.
  • Implemented user authentication and authorization using JWT for secure user sessions.
  • Created validation mechanisms to ensure unique and accurate user registrations.
  • Developed comprehensive CRUD operations for managing blog posts, enabling users to easily create, edit, and delete their travel stories.
  • Integrated dynamic textarea fields to improve the user experience during blog post creation and editing.
